Clique Finder

Clique Finder was the first web application I ever created by myself. Designed with HTML, CSS, PHP and MySQL, Clique Finder allows the user to search and filter through schools in Ohio based upon entered and filtered Criteria.

Drone Delivery Enhancement

Presented at the 2021 Choose Ohio First Poster Conference, this C++ program was created with two other students to increase the success rate of delivery drones by implementing a system manager for a fleet of drones. The manager changes each drones delivery queue with respect to entered parameters such as temperature, humidity, and precipition.

Kent Greek Life Web App

This HTML, CSS, PHP & SQL based web app was designed along with two other students as a part of my Introduction to Database System Design course. It allows for the user to input and search through data regarding Kent State Greek Life in a MySQL Database.
